Dependencies

jQuery

Like any jQuery Plugin, the jQuery library is required for jQuery-Bounce to work. I recomend using the Google CDN of the latest stable. Currently 1.7.1 found here: https://ajax.googleapis.com/ajax/libs/jquery/1.7.1/jquery.min.js

Easing

To achieve a natural bouncing effect, easing is used in the animations. The Easing Plugin is required for jQuery-Bounce to work.
